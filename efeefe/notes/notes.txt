Assertion parameters:
-ea - enable assertions
-da - disable assertions (disabled by default)

eclipse command line parameters to test: 
 - Test	2 = cat
 - Test 1 = Test
 - Testq 3 = Assert Error 1
 - Test 3 = Assert Error 2
 
Note: we are using 1.7 (1.4 and higher java Assert is a key word, 1.3 it is an identifier)

Note: there are combinations you can enable for assertions for just a class, just a package, some combination, or excluding system packages.
